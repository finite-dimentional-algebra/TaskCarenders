# TaskCarenders
タスク管理アプリ (GitHub Pages版)

このアプリは、ブラウザ上で動作するシンプルなタスク管理アプリです。Geminiに作ってもらったやつ。

特徴

インストール不要: URLにアクセスするだけで使用可能。

データ保存: 入力したタスクはブラウザの localStorage に保存されます。

カレンダー表示: 月ごとのタスクを視覚的に確認できます。

リマインド機能: アプリ起動時、一週間後に期限が迫っているタスクがあれば自動で通知パネルが表示されます。

セットアップ方法

GitHubで新しいリポジトリを作成します。

index.html と README.md をアップロードします。

リポジトリの Settings > Pages から、ソースを main ブランチに設定して保存します。

発行されたURLにアクセスすれば完了です。
